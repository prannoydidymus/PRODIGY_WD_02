# Employee Management System (PRODIGY_WD_02)

A state-of-the-art, premium, and fully working Employee Management System built as a Single Page Application (SPA). Designed with modern glassmorphism aesthetics, responsive layouts, client-side data persistence, validation, and a mock authentication portal.

## 🚀 Live Demo Features

- **🔒 Mock Authentication**: Fully protected administration portal (requires login to access employee records).
  - **Admin Credentials**:
    - **Email**: `admin@prodigy.com`
    - **Password**: `admin123`
- **📊 Interactive Dashboard**:
  - Live stats cards (Total Employees, Average Salary, Active vs On Leave percentages, and Department Counts).
  - Custom data visualizations and progress meters.
  - Recent activity tracking log.
- **✨ Complete CRUD Operations**:
  - **Create**: Add new employees with detailed records (avatar selection, contact info, job parameters, status).
  - **Read**: Dynamic responsive datatable with pagination, dynamic query searches, department filters, and active status sorting.
  - **Update**: Secure fields updating with error handling and live-rendered fields.
  - **Delete**: Soft/Hard delete operations using absolute user confirmation modals.
- **💾 Local Storage Persistence**: State is preserved between browser sessions and tab reloads. Pre-loaded with 6 initial professional mock records for demo evaluation.
- **🎨 Glassmorphic Aesthetic**: 
  - Vibrant custom HSL color palette.
  - Dark Mode and Light Mode with micro-animations.
  - Built with responsive design suitable for desktop, tablet, and mobile views.

## 🛠️ Tech Stack

- **Structure**: Semantic HTML5 markup
- **Styling**: Vanilla CSS3 (Custom Variables, Flexbox/Grid, Animations, Backdrop-filters)
- **Logic**: Vanilla ES6+ Javascript
- **Icons**: SVG embedded components for lightweight rendering and vector clarity

## ⚙️ How to Run Locally

1. Clone or download this repository.
2. Open `index.html` directly in any web browser, or serve it using an extension like VS Code's Live Server.
3. Login using the administrator credentials listed above.
